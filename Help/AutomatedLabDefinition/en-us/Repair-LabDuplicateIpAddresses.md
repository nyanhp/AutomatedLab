# Repair-LabDuplicateIpAddresses

## SYNOPSIS
Repair duplicate IPs

## SYNTAX

```
Repair-LabDuplicateIpAddresses
```

## DESCRIPTION
Modifies duplicate IP addresses for machines by incrementing the address by one

## EXAMPLES

### Example 1
```powershell
PS C:\> Repair-LabDuplicateIpAddresses
```

Repair any duplicate IP addresses
