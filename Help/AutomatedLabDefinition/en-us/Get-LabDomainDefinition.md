# Get-LabDomainDefinition

## SYNOPSIS
Returns all definitions of Active Directory domains/forest in the lab

## SYNTAX

```
Get-LabDomainDefinition
```

## DESCRIPTION
Returns all definitions of Active Directory domains/forest in the lab

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabDomainDefinition
```

Return the current domain definition that has been added to a lab.
