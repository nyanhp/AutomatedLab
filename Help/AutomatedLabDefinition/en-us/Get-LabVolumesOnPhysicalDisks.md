# Get-LabVolumesOnPhysicalDisks

## SYNOPSIS
Get lab volumes

## SYNTAX

```
Get-LabVolumesOnPhysicalDisks
```

## DESCRIPTION
Get all file-backed virtual volumes on all disks with drive letter and serial number

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-LabVolumesOnPhysicalDisks
```

Return all physical disks on the lab host
